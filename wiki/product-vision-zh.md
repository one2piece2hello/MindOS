# MindOS 产品愿景

## 人机共享心智

MindOS 的核心目标是：让知识在同一处同时服务人类阅读与 Agent 执行。

### 1. 全局心智同步 — 打破知识孤岛

**痛点：** 传统云笔记割裂、受 API 限制、记录成本高，Agent 很难稳定获取个人深度上下文。

**跃迁：** 一次记录，全局复用。MindOS 提供轻量 Web 捕获与内置 MCP Server，让兼容 Agent 可立即读取你的 Profile、SOP 与项目记忆。

### 2. 透明可控 — 消除 Agent 黑箱

**痛点：** 助手记忆常封闭在系统内部，人类难以审查与纠正 Agent 的中间行为。

**跃迁：** 让思考过程留在明面上。检索、反思、执行结果沉淀为本地纯文本（Markdown/CSV），人类可在 GUI 中持续审计、干预与修正。

### 3. 共生演进 — 让笔记变成执行力

**痛点：** 传统文档系统同步困难，难以成为人机协作任务的执行引擎。

**跃迁：** 知识库即代码。通过 Prompt-Native 记录方式与引用驱动同步，日常笔记可自然转化为可执行指令，让人类与 Agent 在同一心智系统中共同迭代。

## 底层原则：本地优先

所有数据以本地纯文本存储，确保隐私与数据主权，并获得高性能读写体验。

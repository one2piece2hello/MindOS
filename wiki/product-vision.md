# MindOS Product Vision

## Human-AI Shared Mind

MindOS is built around one core idea: your knowledge should be readable by humans and executable by agents in the same place.

### 1. Global Mind Sync — Break Knowledge Silos

**Pain point:** Traditional cloud notes are fragmented, API-gated, and high-friction for capture. Agents cannot reliably access deep personal context.

**Shift:** Capture once, reuse everywhere. MindOS provides lightweight web capture and a built-in MCP server so any compatible agent can immediately use your Profile, SOPs, and project memory.

### 2. Transparent and Controllable — Remove Agent Black Boxes

**Pain point:** Assistant memory is often hidden in closed systems. Humans cannot inspect or correct intermediate agent behavior.

**Shift:** Keep thinking in the open. Retrieval, reflection, and execution outcomes are written to local plain text (Markdown/CSV). Humans can audit, intervene, and correct continuously in the GUI.

### 3. Symbiotic Evolution — Turn Notes into Execution

**Pain point:** Traditional document systems are hard to sync and weak as execution engines for real human-agent collaboration.

**Shift:** Knowledge as Code. With prompt-native writing and reference-driven sync, daily notes naturally become executable instructions. Humans and agents improve each other in one evolving shared mind.

## Foundation: Local-first

All data is stored locally as plain text. This ensures privacy, full ownership, and fast read/write performance.
